# Day 4 - Salesforce Flow Builder

# What is Flow Builder?
Flow Builder is a Salesforce automation tool used to automate business processes without coding.

# Types of Flows
-----------------
# Screen Flow
Used for user interaction with screens and forms.
# Record Triggered Flow
Automatically runs when records are created, updated, or deleted.

# Automation Ideas

1. Automatic Student Registration Confirmation
2. Automatic Fee Reminder
3. Automatic Student ID Generation
4. Faculty Notification for Full Course
5. Attendance Warning Alert
---
# Flow Design Thinking
# Process: Automatic Fee Reminder System
Trigger:
Student fee due date is near
Steps:
1. Check fee due date
2. Verify payment status
3. Send reminder notification
4. Update reminder status

Decision:
Fee paid or not

Final Action:
Student receives reminder

# Manual vs Automated Process

# Manual Process
Staff manually reminds students about fees.

# Problems
- Slow
- Human errors
- Difficult to manage

# Automated Process
Salesforce automatically sends reminders.

# Benefits
- Faster
- Accurate
- Saves time

### Why automation matters in enterprise systems
