# Salesforce Summer Program – Day 2



# Salesforce Platform Basics

Salesforce is a **cloud-based CRM platform** where everything is built using:
- Apps
- Objects
- Tabs

---

# What is an App?
An App is a **collection of objects and functionality** used for a business process.

Example:
- Sales App
- Service App

---

# What is an Object?
An Object is like a **database table** that stores data.

Example:
- Account
- Contact
- Opportunity

---

# What is a Tab?
A Tab is a **user interface link** used to access objects.

Example:
- Account Tab → shows all accounts
- Contact Tab → shows all contacts

---

# CRM → Platform Mapping

From Day 1:
- Account → Object (Company data)
- Contact → Object (Person data)
- Opportunity → Object (Sales data)

👉 These all live inside Salesforce Apps

---

# Configuration vs Coding

# Configuration 
Used when:
- Creating objects
- Adding fields
- Page layout changes
Example:
- Create Student object
- Add Name, Marks fields

---

# Coding (Apex)
used when there is:
- Complex business logic needed
- Automation beyond clicks

Example:
- Auto-assign leads
- Complex validation rules
