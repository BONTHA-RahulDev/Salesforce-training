# Day 8 - Lightning Web Components Basics

## What is LWC?
Lightning Web Components (LWC) is a modern UI framework by Salesforce used to build fast, reusable, and scalable web components using HTML, JavaScript, and modern web standards.

## Why Salesforce Uses LWC
Salesforce uses LWC because it:
- Improves performance
- Supports reusable components
- Uses modern JavaScript standards
- Makes UI development faster and scalable

## My UI Screens (College Management System)
1. Student Registration Form
2. Course Dashboard
3. Attendance Management Screen
4. Faculty Panel
5. Notifications Dashboard

## Component Breakdown (Example: Student Dashboard)
- Header Component → Displays title and navigation
- Student Info Component → Shows student details
- Attendance Component → Shows attendance records
- Notification Component → Shows alerts and updates

## Frontend vs Backend Logic

### Frontend (UI - LWC)
- Button click handling
- Showing data on screen
- Displaying notifications

### Backend (Apex)
- Data validation
- Fee calculation
- Database updates
- Business logic execution

## Why Reusable Components are Important
Reusable components reduce code duplication, improve maintainability, and allow developers to build scalable enterprise applications efficiently.

## Reflection
Modern enterprise systems use component-based architecture because it improves scalability, maintainability, and allows multiple developers to work on independent UI parts without conflicts.
